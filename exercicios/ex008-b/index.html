Código Fonte

Para escrever um código fonte, usamos a tag <code>. É uma fonte monoespaçada ideal para exibir código pois todas as letras possuem a mesma largura.

Por exemplo: System.out.println("Olá, mundo!"), todas as letras terão a mesma largura

Exemplo com texto em Python onde temos as identações presentes:

        
    num = int(input("Digite um número: "))
    if num % 2 == 0:
        print("O número é par.")
    else:
        print("O número é ímpar.")
